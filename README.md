# waifugen
Generate waifus. Lots of waifus.

## Installation Instructions

Requires a fal.ai account and API key

```
git clone https://github.com/lalalune/waifugen
cd waifugen
npm install
cp .env.template .env
paste in FAL_AI_API_KEY=<your api key here>
npm run start
``
